# Intro

The purpose of this sheet is to document the formal definition and requirements
of assembly (GA) language specifications.

# Motivations

GA aims to be writeable easily by humans, and by the Grassroots virtual machine
(GVM). It aims to provide convenience abstractions over the resulting bytecode
(GBC), and namely make it easy to compile to and optimize as an
intermediary language.
